About SRCT-Mobile

SRCT-Mobile provides a modularizable platform for the deployment of self-contained, independent native and web-wrapper mobile applications.

Mobile apps can be easily added and removed from the application database, allowing new developers to quickly get their application off the ground with SRCT sponsorship.

Three Core Principles

Skeleton is built on three core principles:

  • A Responsive Grid Down To Mobile: Elegant scaling from a browser to tablets to mobile.
  • Fast to Start: It's a tool for rapid development with best practices
  • Style Agnostic: It provides the most basic, beautiful styles, but is meant to be overwritten.
